# seer_eyou_client

赛尔号插件管理器（Electron 客户端），用于统一管理与赛尔号微端相关的插件。

## 简介
本项目基于 Electron 开发，提供插件的安装、启用/禁用与更新，配合赛尔号微端实现便捷的启动与扩展。

## 功能
- 插件管理：安装、启用/禁用、更新
- 启动集成：与赛尔号微端协同工作

## 下载与安装
- 赛尔号微端下载地址：[SeerLauncher_setup.exe](https://newseer.61.com/apk/installer/SeerLauncher_setup.exe)
